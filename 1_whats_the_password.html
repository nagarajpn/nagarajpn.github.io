What's the Password??

Posted on November 16, 2014 at 2:37 PM

Today's project is a sequence detector using the Arduino UNO.

Whenever a particular set of characters appear at the serial port of the microcontroller, then the LED blinks 5 times. The characters are sent from the computer to the Arduino using the serial monitor present in the Arduino IDE.
